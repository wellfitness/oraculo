# Claude Code Update Command

Updates Claude Code to the latest version using the official install command.

## Steps

1. Check current version
2. Force reinstall to latest version
3. Verify installation

```bash
# Check current version
claude --version

# Update Claude Code to latest version (force reinstall)
claude install --force

# Verify new version
claude --version
```

This ensures you always have the latest features and bug fixes from Claude Code.
